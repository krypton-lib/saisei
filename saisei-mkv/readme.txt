MKV
---
a multi-platform library for working with mkv files. the `ebml` module is used to read the file and
the `codegen` module is used to generate element declarations to make our lives easier.

Support?
---
feel free to join our discord server: https://discord.gg/8R4d8RydT4

Examples?
---
you can find some code examples in /saisei/src/jvmTest. the current tests include local playback using
javax.sound & discord playback using the kord discord library for the jvm (multi-platform pending)
